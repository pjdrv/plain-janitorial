# Plain Janitorial

Autonomous cleaning services powered by robots and human expertise. Serving Boston, MA and surrounding areas.

## Features

- **Instant Quote Calculator** - 5-step pricing tool for facilities
- **Client Dashboard** - Real-time robot fleet monitoring, scheduling, quality reports
- **Admin Dashboard** - Client management tools
- **Authentication** - Secure login/signup system

## Tech Stack

- Next.js 14 (App Router)
- React + TypeScript
- Tailwind CSS
- Prisma ORM
- NextAuth.js
- Framer Motion

## Getting Started

```bash
# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Edit .env.local with your values

# Generate Prisma client
npm run db:generate

# Run development server
npm run dev
```
